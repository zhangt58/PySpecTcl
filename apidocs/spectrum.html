Spectrum¶

class spectcl.Spectrum(name, conf, data, **kws)[source]¶

Bases: object

Class for spectrum configuration and data.

Parameters

  • name (str) – Name of the spectrum.

  • conf (DataFrame) – Spectrum definition, columns: Type, Parameters, Axes, ChanType.

  • data (DataFrame) – Spectrum contents, columns of x, v or x, y, v.

Keyword Arguments

  • map (bool) – If do coordinate mapping from channel to world, by default is True.

  • client (SpecTclClient) – SpecTclClient instance.
